CodeRorrim

A code parser for iOS and OSX.

Clone on GitHubDonate with PayPal

CodeRorrim is a multi-language parser written in Objective-C. It parses, syntax highlights, and tokenizes. It's powered by CodeMirror.

Many languages supported.

Wanna parse Ruby? Python? JavaScript? CoffeeScript? PigLatin? CodeRorrim has you covered.

Parsing, syntax highlighting, tokenizing.

Have you ever thought to yourself "I wish there was a light-weight, modern parsing engine I could use in my iOS and OSX apps." Well, here you go. CodeRorrim is open-source and ready for you to use. Parses over 30 languages including pig latin.

Easy Syncing

Sync the parser with an NSTextView and parse as you type...live!

API

Use CodeRorrim in your own projects. Pull requests welcome. MIT licensed.
